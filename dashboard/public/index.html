🔍 RAG & Search

          
Controls how memories are retrieved and injected into the agent's context window.

          

            
            
            Number of memories fetched via vector cosine similarity before graph traversal. Higher = more direct
              context but slower.
          

          

            
            
            Maximum total memories injected into context after adding graph-linked results. Cap this to avoid
              overloading the context window.
          

          

            
            
            Boosted similarity score assigned to memories found via graph traversal (linked). Higher = more
              weight given to linked results. (0.0–1.0)
          

          

            
            
            Maximum number of edge hops during recursive graph traversal (A→B→C = depth 2). Higher depth
              discovers distant connections but increases query time.
          

        


        
        

          
🎭 Persona

          
Controls how persona traits are selected and injected into the agent's system prompt.

          

            
            
            Number of context-relevant persona traits injected per prompt (in addition to always-active traits).
              Higher = more identity context.
          

        


        
        

          
📝 System Prompt Templates

          
These blocks are injected into the agent's system prompt. Changes take effect on the next user
            message. Use concise, behavior-guiding language.

          

            
            
            Instructions for how the agent should silently save, update, and link knowledge.
          

          

            
            
            Instructions for when/how the agent modifies its own persona traits.
          

          

            
            
            Autonomous loop behavior. Use {{heartbeatFilePath}} as a placeholder for the path below.
          

          

            
            
            Absolute path to the heartbeat task file the agent checks on each autonomous loop.
          

        


        
        

          
💤 Sleep & Maintenance

          
Thresholds for the background knowledge graph optimization cycle. These run when "Sleep Cycle"
            is triggered.

          

            

              
              
              Merge memories with cosine similarity above this threshold. Higher = stricter dedup (fewer merges).
                Default 0.80.
            

            

              
              
              Archive unused memories older than this many days. Prevents memory bloat over time.
            

            

              
              
              During sleep, only auto-create links between memories with similarity above this floor.
            

            

              
              
              During sleep, skip link creation between memories with similarity above this ceiling (likely
                duplicates).
            

            

              
              
              Number of episodic log entries to consolidate per sleep cycle.
            

            

              
              
              Number of memories to scan for near-duplicates per sleep cycle.
            

            

              
              
              Number of similar memories to consider for link creation per source memory.
            

            

              
              
              Number of candidate pairs to send to LLM for classification in each batch.
            

            

              
              
              Total number of memories to scan for potential links per sleep cycle.
            

            

              
              
              Comma-separated list of memory tiers exempt from low-value archival. Default: permanent,
                stable.
            

          

        


        
        

          
🛠️ Technical

          
Internal settings for deduplication and event processing.

          

            
            
            Number of recent event IDs kept in memory to prevent double-processing. Increase for high-throughput
              agents.
